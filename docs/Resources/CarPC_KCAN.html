BMW X1 (E84) Can-bus Codes
        The following information is derived from the K-CAN interface of a BMW X1 (E84).
Other BMW models (and BMW Mini's) have a similar data packet arrangement but with different CAN-ID's.

The K-CAN 1 is a 100KHz interface bus that links the radio and user controls (E6x, E8x and E9x BMW's). It also contains
some information that is found on the 500Khz PT-CAN (The bus that is connected to the OBD-II connector).
The K-CAN 2 is the updated interface bus as used on 'F' Series BMW's, this now runs at a speed of 500KHz

The information below is gained from passively snooping the Canbus and listening to the standard 'chatter'
of the BMW communication network. This differs from many ODB-II tools that actively request debug data.
For more information on the interface board click here.

Reg = Register type, W = Writeable (the ID can be emulated by an external Can controller, whilst in the car with no adverse affects)
Mini = Tested on a 2013 John Cooper Mini and the register is found to be the same.

Source Information
Kombi = Instrument Cluster.
Multi = Multiple sources.
JBE = Junction Box Electronic Module
CAS = Car Access System
SZL = Steering column switch cluster (The SZL is on the 500KHz F-CAN bus. Data is passed onto the K-can via gateway module.)
CON = idrive Controller
DSC = Dynamic Stability Control
